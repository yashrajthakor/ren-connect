# Build Sponsor Highlights Section for RBN Homepage

## Objective

Create a dedicated **Sponsor Highlights** section on the RBN homepage to recognize sponsors and provide premium business visibility.

This section should be dynamic and fully managed from the Admin Panel.

---

## Homepage Display

Add a new section:

**🤝 Our Sponsors**

Display sponsor cards in a horizontal auto-scrolling carousel.

Requirements:

* Auto-scroll continuously
* Manual left/right navigation
* Infinite loop
* Mobile swipe support
* Display 3–4 sponsors at a time on desktop
* Display 1–2 sponsors at a time on mobile
* Smooth animation
* Pause auto-scroll on hover (desktop)

---

## Sponsor Card

Each card should display:

* Company Logo
* Sponsor Name
* Firm Name
* Business Category
* Short Tagline (optional)
* Contact Number
* Website (optional)

Add CTA:

**View Business Profile**

or

**Contact Sponsor**

Use a premium card design that highlights sponsors without overwhelming the page.

---

## Admin Management

Create a new menu:

**Admin → Sponsor Highlights**

Admin/Super Admin can:

* Add Sponsor
* Edit Sponsor
* Delete Sponsor
* Enable/Disable Sponsor
* Upload Company Logo
* Set Display Order

Fields:

* Sponsor Name
* Firm Name
* Business Category
* Company Logo
* Contact Number
* Website (optional)
* Short Tagline (optional)
* Active Status
* Display Order

---

## Display Logic

* Show only Active sponsors.
* Sort by Display Order.
* Support unlimited sponsors.
* Carousel should automatically continue scrolling regardless of the number of sponsors.

---

## UI Requirements

* Premium business appearance
* Consistent card height
* Responsive layout
* Smooth scrolling animation
* Fast loading
* Mobile-friendly
* Match existing RBN branding

---

## Goal

Create a professional Sponsor Highlights section that provides valuable visibility to sponsors while making the RBN homepage more dynamic, engaging, and community-focused.
